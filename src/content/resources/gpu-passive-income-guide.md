## Who This Is For

If you have a gaming PC with a dedicated GPU that sits idle during work hours or overnight — this guide is for you. No technical background required beyond basic file management. You download a file, run it with your credentials, and it runs in the background.

This is not a primary income source. Set that expectation now. The realistic frame: internet bill contribution, maybe streaming subscription money. The reason to do it isn't the immediate payout — it's that the cost to run it is zero and the network is early.

## The Honest Numbers

Mid-range GPU (RTX 3070/4070): ~$10-40/month. High-end GPU (RTX 4090): ~$40-100/month. Both figures depend heavily on network demand. These are estimates, not guarantees. The network is early-stage, which means demand is growing but not fully mature yet.
