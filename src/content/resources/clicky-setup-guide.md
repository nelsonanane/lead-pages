## What's Inside

This free 3-page guide walks you through installing Clicky and running your first session — even if you've never used a GitHub repo before.

**You'll learn:**
- **What Clicky actually does** — takes a screenshot of your screen when you hold Space to talk, sends it to Claude AI, and talks back while pointing a cursor at the exact button it's describing
- **The 3 API keys you need** — Anthropic, AssemblyAI, and ElevenLabs. All have free tiers. Setup takes under 5 minutes each.
- **The exact permissions to grant** — Screen Recording and Microphone in System Settings. Without these, Clicky can't see or hear you.
- **Your first session, scripted** — the exact questions to ask when you open a new app for the first time

## Who This Is For

You saw the demo. An AI cursor flying across a screen and landing on the exact button it was describing. You want that.

This guide is for anyone who wants to use Clicky but has never cloned a GitHub repo, doesn't know what an API key is, and just wants to hold Space and start asking questions. No technical background required. Just a Mac, 10 minutes, and something you've been meaning to learn.
