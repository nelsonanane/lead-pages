## What's Inside

This guide walks you through setting up the `/repurpose` skill in Claude Code — a plain text file that turns any video URL into 9 platform-ready posts automatically.

**You'll get:**
- **The exact setup steps** — SupaData API key, SKILL.md file location, one command to run
- **Platform cheat sheet** — the right length, tone, and format for LinkedIn, Instagram, and X
- **Quick Mode and Batch Mode** — shortcuts for filming days when you need fast turnaround
- **Voice calibration tip** — how to make the output sound like you, not like AI

The whole setup takes about 10 minutes. After that, any video becomes 9 posts in under 5 minutes.

## Who This Is For

If you're creating short-form video content and posting on more than one platform, you're spending 2+ extra hours rewriting the same ideas in different formats. This skill eliminates that.

It works for any creator who films once and wants reach on LinkedIn, Instagram, and X — without writing three separate posts from scratch every time.

The skill handles YouTube, TikTok, Instagram Reels, X videos, and Facebook. You paste the URL. Claude does the rest.
