If you're building your first app with AI tools — or you've started and got stuck at the database step — this guide is for you. No coding experience required. The guide assumes you have Rork downloaded and an idea for what you want to build.

This is not a theory document. Every step is the actual thing to do, in the order to do it.

## The Core Insight

The stack everyone recommends (Rork + Supabase) works. But Convex is better for this use case because Rork generates TypeScript far more accurately than it generates SQL. That means fewer broken integrations, fewer debugging sessions, and a faster path from idea to working app. This guide shows you exactly how to make the swap.
