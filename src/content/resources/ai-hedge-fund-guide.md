This guide delivers the exact setup process for **virattt/ai-hedge-fund** — the open-source repo with 54,000+ GitHub stars that gives you 14 AI investor agents for free.

- The complete setup walkthrough — prerequisites, install, environment variables, first run

## Who This Is For

If you have savings or investments and want smarter analysis than a Google search gives you — this guide is for you. You don't need a finance background. You need Python installed and an API key. The setup takes 15 minutes.

This is not investment advice. The agents give you 14 institutional frameworks applied to any stock you choose. You make the call. But you're making it with more signal than most retail investors ever get.

## The Core Insight

The key design decision in this repo: the agents don't see each other's output. Buffett's agent doesn't know what Burry's agent said. This prevents groupthink. The most useful signal comes from where the agents split — especially when a contrarian like Burry diverges from consensus. That's what this guide teaches you to look for.
