## What's Inside

agency-agents is a GitHub repo with 147 specialized AI agents — each one built with a defined identity, proven workflow, specific deliverables, and success metrics. It hit 77,000 stars because it solves something real: **generic AI gives generic output**. When Claude doesn't know your role, your standards, or your process, it wings it. That's why the copy sounds corporate, the code feels boilerplate, and the strategy feels shallow.

This guide covers the full picture in one place — setup, activation, which agent to reach for, and how to run multi-agent workflows across a project.

**What's covered:**
- **Install in 3 commands** — clone, copy, verify. Works with Claude Code, Cursor, Aider, Windsurf, and Gemini CLI
- **How to activate any agent** — exactly what to say, and how to chain agents mid-session
- **Task-to-agent lookup** — UI, code review, TikTok content, Google Ads audits, brand voice, SEO, product roadmaps
- **Recommended teams by use case** — Creator, Solo Founder, and Developer starter teams
- **The Whimsy Injector** — the most underestimated agent in the repo, and before/after examples showing what it actually does
- **Power combos** — multi-agent workflows for launches, content batches, paid media buildouts, and new features

Solopreneurs, content creators, developers, and small team leads who want expert-level AI output instead of generalist output. If you've been prompting Claude or ChatGPT for specialized work and rewriting everything it gives you — this is the fix.
