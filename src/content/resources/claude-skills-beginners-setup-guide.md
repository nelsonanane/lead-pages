## What's Inside

This 3-page guide breaks down exactly how Claude Skills work and how to get your first one running — even if you've never opened a terminal before.

**You'll learn:**
- **What a Skill actually is** — a plain text file with instructions. That's it. No Python, no JavaScript, no cloud setup.
- **The minimum viable skill template** — copy it, paste it, run it. Claude handles the rest.
- **Why Skills beat regular prompting** — Claude remembers the instructions permanently, so you never repeat yourself again.
- **3 starter skills** — a summarizer, an email drafter, and a content idea generator. All ready to drop in.

## Who This Is For

You're not a developer. You use Claude regularly but you're still manually typing the same kinds of prompts over and over — for emails, for content, for research. You saw someone run a slash command and Claude just *did the whole thing*, and you want that.

This guide is your starting point. One setup, one command, and Claude has a permanent job description it follows every time.
